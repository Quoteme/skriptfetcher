# SkriptFetcher

A simple script to fetch the model theory scripts provided by the amazing [Prof. Dr. Immanuel Halupczok](https://www.math.hhu.de/lehrstuehle-/-personen-/-ansprechpartner/innen/lehrstuehle-des-mathematischen-instituts/lehrstuhl-fuer-algebra-und-zahlentheorie/team/prof-immanuel-halupczok) at the [Heinrich-Heine-University](https://www.hhu.de/) in DÃ¼sseldorf.

Prof. Dr. Halupczok frequently updates his scripts on his personal website. With this little program all these scripts can be downloaded and saved in a local directory for convenient offline access.
